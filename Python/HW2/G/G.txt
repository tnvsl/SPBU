Задача E:

Необходимо написать функцию, которая принимает два отсортированных массива и возвращает результат их объединения. При этом, если элемент второго массива уже содержится в первом, то в объединение такой элемент не попадает.

Дополнительно сортировать массив нельзя.

Пример: 

>>> a = [1, 1, 2, 5]
>>> b = [1, 3, 4, 5, 8]
>>> solution(a, b)
[1, 1, 2, 3, 4, 5, 8]
