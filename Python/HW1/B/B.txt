Задание B:

Напишите функцию, возвращающую строку, вывод которой позволяет нарисовать n пингвинов.

Пингвин:
   _~_   
  (o o)  
 /  V  \ 
/(  _  )\
  ^^ ^^  

Пример:
>>> n = 3
>>> print(solution(n))
   _~_      _~_      _~_   
  (o o)    (o o)    (o o)  
 /  V  \  /  V  \  /  V  \ 
/(  _  )\/(  _  )\/(  _  )\
  ^^ ^^    ^^ ^^    ^^ ^^  

>>> n = 0
>>> print(solution(n))

Примечания:
1) Не используйте циклы
2) Выстраивание пингвинов идет построчно, а не попингвинно
